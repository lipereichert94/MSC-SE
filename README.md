# MSC-SE

Project developed while completing the Master Degree Program in Computer Science. It's a final project to demonstrate the use of CI/CD by DEVOPS teams. 
Here we have a Spring Boot default API that once is updated and need to be deployed in environment wheter Develop profile, Test profile or Production profile, there's a job on Jenkins responsible for execute a pipeline to run the Tests, create new Docker Images and run it through containers automating the processes. 

Technlogies used:
Jenkins + Java Spring Boot + Git + Docker
